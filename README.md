# AI Product Description Generator

An AI-powered web application that helps food processing businesses generate professional product descriptions for e-commerce platforms.

## Tech Stack

- Frontend: React.js
- Backend: FastAPI
- Database: MongoDB Atlas
- AI API: Google Gemini API

## Features

- AI Product Description Generation
- Product Title & Tagline Generation
- SEO Keyword Suggestions
- Tone-Based Content Customization
- Content Regeneration & Refinement
